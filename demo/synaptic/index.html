Synaptic Web Collection
Ten interactive explorations of neural signal propagation, memory formation, fatigue mechanics, and emergent network behavior. Each demo runs at 60fps on modest hardware.

Minimal Pulse

Stripped-down hex ring network. Pure line art with gold signal propagation. No glow, no gradients -- just clean geometry.

minimal / hex-rings / signals

Cascade

Top-to-bottom waterfall layout. Signals flow downward like gold rain through layered rows. Click at the top to trigger cascading waves.

waterfall / gravity / rain

Heartbeat

Center node pulses at 72bpm, sending rhythmic waves outward through concentric rings. The whole network breathes. Click to add extra beats.

rhythm / pulse / breathing

Memory Trace

Signals permanently strengthen edges they traverse. Frequently-used paths become gold highways while unused connections stay dark. Watch memory form.

hebbian / memory / pathways

Fatigue Visualizer

Visible health arcs around each node deplete with every firing. Exhausted nodes turn red and go dark. Recovery is gradual and visible. Live stats in corner.

fatigue / health / recovery

Thought Propagation

Colored thought-pulses travel through a brain-shaped network leaving trails. When thoughts collide at a node, they flash and spawn new directions.

thoughts / collision / multicolor

Rhythm Grid

A step sequencer built from neurons. Click nodes to toggle them on/off. A sweeping playhead triggers tones via Web Audio API. Different rows = different frequencies.

sequencer / audio / interactive

Constellation

Stars drift and self-organize toward hex grid formations via spring forces. Edges form and break dynamically. Click to add disruption. Slow and meditative.

self-organizing / springs / meditative

Dual Brain

Two hemispheres connected by a corpus callosum bridge. Left brain is orderly hex grid, right brain is chaotic. Signals change color when crossing.

hemispheres / bridge / cross-talk

Evolution

Starts with 5 nodes. The network grows, strengthens active paths, and prunes unused connections. New nodes appear near your cursor. A living network.

growth / pruning / alive
